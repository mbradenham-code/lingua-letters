Safety is not an afterthought
Every design decision in Lingua Letters starts with one question — is this safe for students? Here's what that looks like in practice.
🤖 AI reviews every letter
Before any letter is delivered it passes through multiple AI safety checks.
🚩 Flagged = never sent
A flagged letter is held immediately and never reaches the recipient.
👁 Teachers see everything
School account teachers can read all letters in their class at any time.
📱 Parents approve connections
No pen pal connection is active until a parent approves it for under-18 accounts.
✏ Students can explain
When flagged, students can write a 2-sentence explanation. AI and adults decide together.
🔒 Strict flag rules
Teachers and parents each set their own rules. When they conflict, the stricter one always wins.
